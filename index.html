Frequently Asked Questions

To create your first packing list:

  1. Tap the "+" button in the top right corner of the main screen
  2. Enter a name for your list (e.g., "Weekend Trip")
  3. Optionally, choose a template to pre-fill your list with common items
  4. Tap 'Save' to create the list

To add items to a list:

  1. Tap on a list from the main screen to open its details
  2. Tap the "+" button in the top right corner
  3. Enter the item name and quantity
  4. Tap 'Add Item'

To mark items as packed:

  1. In the list detail view, tap the checkbox next to an item
  2. The item will be marked as packed and crossed out
  3. Tap again to unmark if needed

StockStride offers powerful search and filtering:

  • Search: Use the search bar to find lists by name or items within lists
  • Filter: Tap the filter icon to filter by completion status (All, Completed, In Progress, Not Started)
  • Sort: Tap the sort icon to sort by date, name, or progress

To change the theme color:

  1. Tap the gear icon in the top left of the main screen
  2. In the 'Appearance' section, choose your preferred theme color
  3. The change will be applied immediately throughout the app

To share a packing list:

  1. Long-press on a list card from the main screen
  2. Select 'Share' from the context menu
  3. Choose to share as text or PDF
  4. Select your preferred sharing method (Messages, Email, etc.)

To duplicate a list:

  1. Long-press on a list card from the main screen
  2. Select 'Duplicate' from the context menu
  3. A copy of the list will be created with "(copy)" added to the name

To delete a list:

  1. Long-press on a list card from the main screen
  2. Select 'Delete' from the context menu
  3. Confirm the deletion in the popup dialog
  4. Alternatively, you can edit a list and use the "Delete List" button in the Danger Zone

About StockStride

StockStride is a modern, elegant packing list app designed to help you never forget important items on your trips. Whether you're planning a business trip, vacation, or weekend getaway, StockStride makes packing organized and stress-free.

Key Features:

  • Unlimited smart packing lists
  • Real-time progress tracking
  • Powerful search and filtering
  • Beautiful, customizable themes
  • Ready-made templates
  • PDF export for printing
  • Share lists with others
  • Detailed analytics and statistics
  • Achievement system
  • Completely free to use
